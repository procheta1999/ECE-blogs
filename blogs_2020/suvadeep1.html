Android App Development


Suvadeep Maity

Android software development is the process by which new applications are created for the Android operating system. Applications are usually developed in Java programming language using the Android software development kit (SDK). But now we can use Kotlin or Flutter to make Android Apps

What is Android App Development?

Android app development is, to put it simply, the process of creating or developing an application that could function under the Android operating system by Google. Mainly we can use android apps to make our life easy and comfortable.

What is required for Android app development?

We need a Laptop/PC.

Operating System: Mac/Windows/Laptop

RAM: 4gb+ (Required)

 We need an Android Phone also for this.

What is the programming language for Android?

• Language: JAVA/FLUTTER/KOTLIN

Why we use Java for this?

There are many types of features of Java:

Use it in Back-end (Spring-framework)

Use Java in GUI also.

Can strengthen his core concept

Also use Front-End: XML or Drag & Drop.
