Introduction to Web Development/h1>

Arunima Maiti

For developing or designing a website on the internet, you need to learn web development. This course is all about creating your web page on whatever topic you want. Among professionals, web development is the non-designing aspects of building a website, which includes writing markup and coding.


Web Development Tools

To learn web development, you have to know about some essential tools. Web development tools are also known as dev tools, which help a developer to test and debug their code. There are some web browsers like Google Chrome, Firefox, Internet Explorer, Safari, Opera, which provide built-in tools for developers. Web development tools help developers to work in various fields of technology like HTML, CSS, JS, React, amp, Bootstrap, PHP and other components. These equipment are designed and handled by browsers. Popular web browsers also included some extra features to fulfil the increasing demand of developers.


Category of Web Development

Web development is of three types-

1. Front-end Web Development: Every front-end developer must have a basic knowledge of HTML, CSS and JS. They can create a responsive website for all type of modern web development. When you complete all the basic requirements of front-end developing, then learn how to use maps, fonts, icons in HTML. On JS, you should learn how to access the HTML DOM and also how to use AJAX, JSON. After all, these, look forward to some frameworks, learn angular JS, jQuery, React, Vue, W3.

2. Back-end Web Development: At the beginning of back-end developing, you have to learn full-stack SQL, PHP, ASP, Python. After these, you should start learning full-stack JS, including SQL, Node, MySQL, Mongo.db.

3. Full Stack Web Development: In full-stack web development, you should know all the topics of front-end and back-end development. You have to cover all the topics step by step and absorb them.



Short Description of Some Basic Topics

HTML: HTML stands for Hyper Text Markup Language. It is a standard markup language to display designed documents in the browser. HTML builds the structure of a webpage. With HTML, you can include different types of input.

CSS: The full form of CSS is Cascading Style Sheets. It provides different fonts, colours, spacing, etc. CSS can control multiple webpage layout at a time. It is independent and can be used in different frameworks and markup languages.

JS: JavaScript is a programming language also used in web development. It can construct and update HTML, CSS. JavaScript is used explicitly as an object-orientated programming language.

Bootstrap: Bootstrap is a framework for designing and creating websites faster and easier. It includes HTML, CSS, JS. This framework has different types of typography, buttons, table, forms, modals, navigations, carousels, etc. Bootstrap is a consistent framework supporting all the major browsers.


Conclusion

In the present scenario of the industry, web development is a much preferable course. It is a great course to start your technical journey.
