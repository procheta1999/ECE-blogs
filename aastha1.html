From primary classes I used to know basic ideas about computers, it's parts and stuff. As little time passed, I heard about the word programming language. Blue J, my first programming language which I studied. After some time, I saw my elder brother running Java programs then I came to know about word coding whose definition basically means creating instruction for computers using programming language.

After taking admission in NSEC my love for coding increased. Then I knew this coding is used in every field of technology whether developing software or for travel tourism and aviation and so on. If we are working or preparing for corporate jobs then coding is like the first step.



Just like architecture, which is an art used by people daily and affects their everyday life, so is code .We can create something and people can interact with ,use , touch and work with .Unlike physics, where physical restraints are in everything we do ,there are no limitations in coding beside our ability to think of a logically consistent solution.

The sense of accomplishment when solving a really complicated problem cannot be beat! It just feels as if we have been at the best party in the town.
